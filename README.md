# QuizApp
Android quiz application which stores it's questions in an SQLite database.

![Zrzut ekranu 2020-05-19 o 11 55 23](https://user-images.githubusercontent.com/48799512/82312774-cde37900-99c7-11ea-8bee-a492cb687c6a.png)

## Description

This is a quiz application on the android system with the option of choosing a category of questions. It is also possible to add / remove questions from the database.

## Technologies

- Android SDK
- SQLite
- MVVM
